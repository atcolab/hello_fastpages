SQL Order of execution

The processing of a SQL statement:

  • from :pick the tables to be queried> - where :filter the rows> - group by :aggregate rows together> - having :filter the aggregates> - select :select the columns that appear in the output> - order by :sorts the rows> - limit :limit the number of return rows F-W-GB-H-S-OB-L
%%sql

select title, rental_rate, replacement_cost, ceil(replacement_cost / rental_rate) as break_even
from film
order by ceil(replacement_cost / rental_rate) desc
limit 5;
